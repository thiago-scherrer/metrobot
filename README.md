# metrobot

Parseador de status do metro de São Paulo. O tempo de parser é de 1 em 1 minuto

## requerimentos

go > go1.11.5

## instalação

```bash
go get github.com/thiago-scherrer/metrobot
```

## instalação manual

Build

```bash
go build
```

Execução

```bash
./metrobot
```
